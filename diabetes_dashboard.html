Project Documentation

📋 Project Overview

This project builds a machine learning pipeline to predict diabetes in patients using clinical diagnostic data from the Pima Indians Diabetes Dataset. It covers the full data science lifecycle: data exploration, preprocessing, feature analysis, model training, evaluation, and deployment.

🗃️ Dataset

The Pima Indians Diabetes Dataset contains 768 records of female patients of Pima Indian heritage. Each record contains 8 diagnostic features: Number of pregnancies, Glucose concentration, Blood pressure, Skin thickness, Insulin level, BMI, Diabetes pedigree function, and Age. The target variable is binary — 1 for diabetic, 0 for non-diabetic (34.9% positive class).

⚙️ Methodology

The pipeline includes: (1) EDA — distribution analysis, correlation study, class balance check. (2) Preprocessing — missing value imputation, StandardScaler normalization for LR. (3) Model Training — Logistic Regression, Random Forest (100 trees), Gradient Boosting (100 estimators), all with 80/20 stratified train-test split and 5-fold cross-validation. (4) Evaluation — Accuracy, AUC-ROC, Precision, Recall, F1-Score, Confusion Matrix.

🏆 Results

Random Forest achieved the best performance: 90.3% accuracy, 95.4% AUC-ROC, and 89.5% average cross-validation accuracy. Glucose level and Age were identified as the most important predictors. The model generalizes well with low variance across cross-validation folds (±2.6%).
